Auxiliary function, gets the path to a registry key and returns its number of sub-keys or values.

Parameters

  • path (str) – url containing path to a registry key, must be decoded using app.get_key_subkey

  • check (int) – if 0, function returns the number of sub-keys, if 1 function returns the number of values

Returns

number of sub-keys or values

int
