app.find_string(param)[source]

Recursively searches for string in registry.

Receives the path to a registry key, a value name and a string to be searched. The key and value are the starting point.

  1. Finds the value’s index inside the key.

  2. Calls recursive_search, ignoring the first value_index values.

  3. If the search was not successful, finds index of searched key inside parent key.

  4. Recursively searches parent key, ignoring the first key_index sub-keys.

  5. If the search keeps failing until it reaches the root of one of the main keys, it moves on to the next root key.

  6. Repeats the process until it finds a match or reaches the end of the registry.

Returns either the path to a key and value or an empty string.

Parameters

param (str) – url string containing path to the search starting point (registry key and value)

Returns

json string containing path to first occurrence, or an empty string if there are no matches

Return type

str

app.recursive_search(base_key, sub_key, search_string, start_key=0, start_value=0)[source]

Recursively searches for string in registry key.

Searches for search_string in values starting from start_value, and then in sub-keys starting from start_key.

Parameters

  • base_key (handle) – handle for a registry key

  • sub_key (str) – name of sub-key of handle

  • search_string (str) – string that must be found

  • start_key (int) – index of first key that will be searched, by default 0

  • start_value (int) – index of first value that will be searched, by default 0

Returns

path to key and value if search_string is found, or an empty string otherwise

Return type

str
